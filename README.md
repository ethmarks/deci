# deci

`deci` is a text editor like `nano`

'[Deci](https://en.wikipedia.org/wiki/Deci-)' is the Metric prefix for 10^-1, like how [Nano](https://en.wikipedia.org/wiki/Neci-) is the prefix for 10^-9.

- `deci` is pretty small, but `nano` is *tiny*.
- `deci` is not very well known, while `nano` is extremely common.

Written in [Go](https://go.dev/) with [Bubble Tea](https://github.com/charmbracelet/bubbletea).
